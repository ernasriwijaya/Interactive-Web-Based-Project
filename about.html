  • What is this project about?

    We develop this technology with an interactive web-based program which focus on Digital Wellness Awareness for youth. There are many existing mobile applications that focus on digital wellness on play store or app store. Meanwhile, existing web-based program for digital wellness is hard to find. This web-based program will perform a simple task for youth such as provide an interactive activities or quiz. Moreover, website is an essential tool for spreading awareness for digital wellness.


  • Why we develop this project?

    The main cause that influenced this project is lack of awareness, interest and involvement of youth in Digital Wellness. To solve this issue, we have to develop this website that we believe can help us in spreading awareness.
